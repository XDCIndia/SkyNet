## xdc-node-setup (SkyOne) Security Findings

### Critical (P0)

#### 1. RPC Endpoint Exposure
**Issue:** RPC port bound to 0.0.0.0 in some configurations  
**Impact:** Remote attackers could access node RPC, potentially steal funds  
**Status:** Mitigated in recent commits - now defaults to 127.0.0.1  
**Remediation:**
```bash
# Ensure RPC is bound to localhost only
RPC_ADDR=127.0.0.1
```

#### 2. CORS Wildcard Configuration
**Issue:** CORS_DOMAIN defaults to "*" in some templates  
**Impact:** Any website can make RPC calls to the node  
**Status:** Fixed in docker/mainnet/.env.example  
**Remediation:**
```bash
# Restrict CORS to specific origins
RPC_CORS_DOMAIN=http://localhost:7070
```

### High (P1)

#### 3. pprof Endpoint Exposure
**Issue:** pprof debugging endpoint may be exposed  
**Impact:** Information disclosure, potential DoS  
**Status:** Addressed with PPROF_ADDR=127.0.0.1  
**Remediation:** Remove pprof from production builds

#### 4. Docker Socket Mount (Historical)
**Issue:** Docker socket was previously mounted into containers  
**Impact:** Container escape vulnerability  
**Status:** Fixed - mount removed by default  
**Remediation:** Use Docker Monitor profile only when necessary

#### 5. Privileged Containers (Historical)
**Issue:** Containers previously ran with elevated privileges  
**Impact:** Full host access if compromised  
**Status:** Fixed - no-new-privileges enabled  
**Remediation:** Continue using security_opt and cap_drop

## XDCNetOwn (SkyNet) Security Findings

### Critical (P0)

#### 1. Secrets in Git History
**Issue:** API keys and credentials previously committed  
**Impact:** Full system compromise if keys not rotated  
**Status:** Fixed in commit 9969072  
**Remediation:**
```bash
# Rotate all exposed secrets
# Update API_KEYS environment variable
# Regenerate database credentials
```

#### 2. Legacy API Authentication Bypass
**Issue:** Legacy endpoints lack authentication  
**Impact:** Unauthorized node registration/deletion  
**Status:** Partially addressed  
**Remediation:** Add authentication middleware to all legacy routes

### High (P1)

#### 3. Insecure API Key Generation (Historical)
**Issue:** API keys previously generated with Math.random()  
**Impact:** Predictable keys could be brute-forced  
**Status:** Fixed - now uses crypto.randomBytes  
**Remediation:** Rotate all existing API keys

#### 4. Unbounded Data Growth
**Issue:** No data retention policy for metrics  
**Impact:** Performance degradation, potential DoS  
**Status:** Issue created (#387)  
**Remediation:** Implement tiered retention strategy
